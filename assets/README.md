# Assets Directory

Direktori ini berisi semua file aset yang digunakan oleh landing page.

## File yang Diperlukan

- **avatar.jpg** - Foto profil Anda (Rekomendasi: 200x200px atau lebih, format JPG/PNG)
- **resume.pdf** - File resume/CV Anda (Opsional, untuk link download)

## Struktur

```
assets/
├── avatar.jpg
├── resume.pdf
└── icons/
    └── (SVG custom icons jika diperlukan)
```

## Catatan

- Avatar akan ditampilkan dalam bentuk lingkaran di halaman utama
- Letakkan file dengan nama yang sama seperti di index.html untuk menghindari error 404
- Pastikan ukuran file tidak terlalu besar untuk performa optimal
