🎨 Spesifikasi Desain (UI/UX)

Dokumen ini menjelaskan panduan desain visual untuk landing page me.sanopalz.my.id. Tujuannya adalah menjaga konsistensi tampilan agar terlihat profesional dan elegan.

1. Konsep & Tema

Gaya Visual: Minimalis, Bersih, dan Modern.

Fokus Utama: Keterbacaan (Readability) dan kemudahan akses tombol tautan.

Tema Warna: Mendukung Light Mode (terang) dan Dark Mode (gelap) secara otomatis mengikuti pengaturan sistem pengguna.

2. Palet Warna (Color Palette)

Dark Mode (Default):

Background: #121212 (Hitam pekat/abu-abu sangat gelap)

Card/Tombol: #1E1E1E (Abu-abu gelap untuk kontras)

Teks Utama: #F5F5F5 (Putih tulang / Off-white)

Teks Sekunder: #A0A0A0 (Abu-abu terang)

Aksen (Hover/Link): #3B82F6 (Biru elegan, warna identitas)

Light Mode:

Background: #F9FAFB (Abu-abu sangat terang)

Card/Tombol: #FFFFFF (Putih murni)

Teks Utama: #111827 (Abu-abu kehitaman)

Teks Sekunder: #4B5563 (Abu-abu sedang)

Aksen (Hover/Link): #2563EB (Biru sedikit lebih gelap)

3. Tipografi

Kita menggunakan font sans-serif yang modern dan mudah dibaca di berbagai resolusi layar.

Font Family Utama: Inter, Roboto, atau sistem UI default (Apple System, Segoe UI).

Ukuran (Desktop/Mobile):

Nama / Header: 24px - 28px (Bold)

Deskripsi / Bio: 16px (Regular)

Teks Tombol: 16px (Medium)

4. Struktur Tata Letak (Layout)

Layout menggunakan pendekatan satu kolom (Single Column) yang terpusat di tengah layar (Center Aligned), sangat ideal untuk tampilan mobile.

Header / Profil:

Foto Profil (Avatar) berbentuk lingkaran (100x100px atau 120x120px).

Nama ("Sanopalz").

Label profesi / Bio singkat (misal: "Web Developer & Tech Enthusiast").

Konten / Tautan:

Kumpulan tombol tautan yang berjejer ke bawah (Stack).

Setiap tombol memiliki ikon (SVG) di sebelah kiri dan teks di tengah.

Footer:

Teks kecil untuk hak cipta atau kredit (contoh: "© 2026 Sanopalz").

5. Aset Ikon

Menggunakan ikon SVG yang ringan dan scalable. Direkomendasikan menggunakan pustaka ikon:

Lucide Icons atau Phosphor Icons.
