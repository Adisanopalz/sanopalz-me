🌐 me.sanopalz.my.id

Repositori ini berisi kode sumber dan dokumentasi untuk landing page personal (berfungsi sebagai alternatif Linktree/Portofolio singkat) yang di-host pada subdomain me.sanopalz.my.id.

📌 Tentang Proyek Ini

Proyek ini dibangun untuk menjadi "kartu nama digital" yang modern, cepat, dan responsif. Pengunjung dapat melihat profil singkat, proyek terbaru, dan tautan ke berbagai platform media sosial saya.

🚀 Fitur Utama

Cepat & Ringan: Dibangun tanpa framework berat untuk memastikan waktu muat (load time) yang instan.

Responsif: Tampilan yang menyesuaikan dengan sempurna di layar HP, tablet, maupun desktop.

Tautan Tersentralisasi: Menyimpan semua tautan penting (GitHub, LinkedIn, Instagram, dll) dalam satu halaman.

Dukungan Dark Mode: Menyesuaikan dengan preferensi tema sistem perangkat pengunjung.

📂 Struktur Dokumentasi

Untuk detail lebih lanjut mengenai proyek ini, silakan baca dokumen berikut:

🎨 Spesifikasi Desain (design.md): Berisi panduan warna, tipografi, dan struktur antarmuka pengguna (UI).

💻 Dokumentasi Developer (developer.md): Berisi informasi tentang tech stack, cara instalasi, dan panduan deployment.
