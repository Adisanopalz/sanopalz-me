# 🤝 Kontribusi

Dokumentasi ini memandu cara berkontribusi pada proyek me.sanopalz.my.id.

## Cara Mengontribusi

1. Fork repositori ini.
2. Buat branch baru untuk fitur atau perbaikan Anda:
   ```bash
   git checkout -b feature/nama-fitur
   ```
3. Lakukan perubahan dan commit dengan pesan yang jelas:
   ```bash
   git commit -m "Tambah: deskripsi fitur"
   ```
4. Push ke branch Anda:
   ```bash
   git push origin feature/nama-fitur
   ```
5. Buat Pull Request dengan deskripsi lengkap.

## Standar Kode

- **HTML**: Gunakan semantic HTML5 dan pastikan accessibility.
- **CSS**: Ikuti variable naming dalam style.css, pastikan responsive.
- **JavaScript**: Gunakan vanilla JS, hindari jQuery atau library besar.

## Testing

Sebelum membuat PR, pastikan:

- Halaman responsive di mobile, tablet, dan desktop.
- Dark mode dan light mode berfungsi dengan baik.
- Semua link eksternal terbuka di tab baru.
- Tidak ada error di browser console.

## Pertanyaan & Diskusi

Jika memiliki pertanyaan atau ingin diskusi tentang fitur baru, buka GitHub Issue.
